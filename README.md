# Config your Mac
When you get a new MacBook, you always need to pair it with some software to assist with work, study, and even entertainment. Here is a script `mac_config.sh` to automatically configure macOS. It is recommended to create a directory `Scripts/` specifically for storing personal scripts.

![MacOS+Homebrew](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTS8gIW6inYThqKjaZRmXg-rBM31Z2CrRXztg&s)

## Files

- mac_config.sh
- myprofile: my own configuration, as a supplement to `.zshrc` and `.bash_profile`
- install_skills.sh: to install skills from https://www.skills.sh/ (for OpenCode)
- pin.sh: install commonly used python 3rd lib by pip (globally)
